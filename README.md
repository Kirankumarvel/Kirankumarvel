[![Repo](https://img.shields.io/badge/GitHub-PulseCheck-1F8EF1?style=flat-square&logo=github)](https://github.com/Kirankumarvel/PulseCheck)
![AWS EC2](https://img.shields.io/badge/AWS-EC2-FF9900?style=flat-square&logo=amazon-ec2)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat-square&logo=docker&logoColor=white)
![GitHub Actions](https://img.shields.io/badge/GitHub_Actions-2088FF?style=flat-square&logo=github-actions&logoColor=white)

> Every push triggers build → containerize → deploy to AWS EC2 (no manual steps).

```text
git push
  → GitHub Actions
  → Docker build
  → SSH to EC2
  → pull & run container
  → ✅ live in ~90 seconds
```

| What it proves | Details |
|---|---|
| 🔄 End-to-end CI/CD | GitHub Actions → Docker → SSH → EC2 |
| 🐳 Containerization | Dockerized FastAPI + health checks |
| ☁️ Cloud deployment | AWS EC2 (Free Tier) |
| 🔐 Secrets | GitHub Secrets (keys/credentials) |
